## 🤖 What Is This?

**Opencode Bob** is a fully persistent, self-aware AI assistant built on Claude's architecture patterns. It runs locally on Windows with full computer control capabilities.

## ✨ Features

- **🧠 Persistent Memory** - Remembers everything across sessions
- **🦾 Full Computer Control** - Mouse, keyboard, apps, files, processes
- **⚡ Parallel Execution** - Unlimited parallel agents for max speed
- **🧪 Auto-Learning** - Learns from every interaction
- **🎯 Intent Classification** - Automatically routes requests
- **💾 Dream Engine** - Autonomous memory consolidation
- **🏆 Reward Scoring** - Tracks performance and evolves patterns

## 🚀 Quick Start

```powershell
# Start Bob
pwsh -File C:/Users/clayt/opencode-bob/start-bob.ps1
```
