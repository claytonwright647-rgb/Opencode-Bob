# Intent Template: TRIVIAL

## Description
Simple requests requiring no planning or analysis

## Route
Direct execution

## Keywords
- what time is it
- what is the date
- show me
- list
- what does
- how do i
- just
- quick
- simple
- check
- get
- show
- tell me

## Behavior
1. Acknowledge request briefly
2. Execute directly
3. Return result

## Examples
- "what time is it?" → Return current time
- "list my files" → List directory
- "show my tasks" → Show task list
