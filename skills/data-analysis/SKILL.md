# Data Analysis Skill

Analyze data and generate insights.

## Capabilities
- Statistical analysis
- Data cleaning
- Visualization suggestions
- Pattern detection
- Correlation analysis

## Process
1. Load and explore data
2. Clean missing/invalid values
3. Calculate statistics
4. Detect patterns
5. Generate insights

- Columns: Y
- Missing: Z%

| col1 | X | Y | Z |
