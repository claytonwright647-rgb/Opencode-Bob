# Documentation Generation Skill

Generate comprehensive docs for any codebase.

## Output Formats
- **README.md** - Project overview
- **API.md** - Function/class documentation
- **CHANGELOG.md** - Version history
- **CONTRIBUTING.md** - Contribution guidelines

## README Template
```markdown
# Project Name

Brief description (1-2 sentences)

## Features
- Feature 1
- Feature 2
- Feature 3

## Installation
\`\`\`bash
npm install project-name
\`\`\`

## Usage
\`\`\`javascript
const example = require('project-name');
// Example code
\`\`\`

## API

### functionName(params)
Description

**Params:**
- `param1` (type): Description

**Returns:** (type)

## License
MIT
```
