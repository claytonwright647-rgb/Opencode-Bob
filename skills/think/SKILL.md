# Deep Thinking Skill

Systematic reasoning for complex problems.

## Thinking Framework

### 1. DEFINE
- What exactly are we solving?
- Success criteria?
- Constraints?
- What have we tried?

### 2. DECOMPOSE
- Break into independent parts
- Identify dependencies
- Find the critical path
- Note what can parallelize

### 3. EXPLORE
- Gather relevant information
- Research patterns/solutions
- Consider multiple approaches
- Weight tradeoffs

### 4. REASON
- Walk through logic step-by-step
- Consider edge cases
- Check for assumptions
- Look for edge cases

### 5. DECIDE
- Choose best approach
- State why others rejected
- Note confidence level
- Define checkpoint criteria

### 6. PLAN
- Break into executable steps
- Note dependencies
- Estimate complexity
- Define rollback
